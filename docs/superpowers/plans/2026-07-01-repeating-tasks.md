# Repeating Tasks Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let a task repeat on a schedule (every N days / weekly / monthly, with an optional end date), rolling over until done, waiting in the stash between occurrences, and keeping a per-task completion history.

**Architecture:** A repeating task is a single `TaskItem` carrying a JSON-encoded `RecurrenceRule`. Open repeats survive daily cleanup for free (cleanup only purges *done* tasks). When a repeat is finalized as done, it isn't deleted — it appends to its own completion history and reschedules to its next occurrence, either re-opening in place (daily) or hopping into the stash with a return date (weekly/monthly), reusing the existing `isStashed` + `stashReturnDate` + `StashReturn` machinery. The stash sheet splits into "Stashed" (one-off) and "Repeating" sections. A recurrence editor sheet is reached via a long-press "Repeat…" context menu.

**Tech Stack:** Swift, SwiftUI, SwiftData (CloudKit-backed via App Group), XCTest. Project generated by XcodeGen (`project.yml`).

## Global Constraints

- SwiftData model is CloudKit-backed: every new stored property MUST be optional or have a default value (lightweight migration). — from spec "Data model".
- Recurrence is **day-granular** — no time-of-day anywhere. — from spec "Guiding principle".
- All computed occurrence dates are `calendar.startOfDay(...)`, matching the stash's date-boundary model. — from spec "Recurrence rule".
- Weekdays use `Calendar` numbering: **1 = Sunday … 7 = Saturday**. Month days are **1…31**.
- Pure logic lives in `Shared/` value types/enums (testable without SwiftData), following the `DailyCleanup` / `StashReturn` / `StashDuration` pattern. — from spec.
- Tests are XCTest, using an in-memory `ModelContainer` for context-based tests and fixed `Calendar`s (UTC or a named zone) for date-based tests, matching `StashTests` / `DailyCleanupTests`.
- Version bump lives in `project.yml` (`MARKETING_VERSION`, `CURRENT_PROJECT_VERSION`), never the `.pbxproj` (XcodeGen regenerates it).
- Test command (adjust the simulator name if needed):
  `xcodebuild test -scheme DailyTodo -destination 'platform=iOS Simulator,name=iPhone 16'`

```swift
import Foundation

/// How often a task repeats. Day-granular; no times.
/// Weekdays use Calendar numbering (1 = Sunday … 7 = Saturday); month days are 1…31.
enum Frequency: Codable, Equatable {
    case everyNDays(Int)      // N >= 1; 1 means every day
    case weekly(Set<Int>)     // set of weekdays
    case monthly(Set<Int>)    // set of days-of-month
}

/// A repeat schedule plus an optional inclusive end date (`nil` == forever).
/// Pure value type — all the recurrence math lives here so it's unit-testable
/// without SwiftData, matching `StashDuration` / `DailyCleanup`.
struct RecurrenceRule: Codable, Equatable {
    var frequency: Frequency
    var endDate: Date?

    /// Fixed English short weekday names, indexed by Calendar weekday (1 = Sun).
    /// Deliberately not locale-derived, so `summary` is deterministic and testable.
    static let weekdayShortNames = ["", "Sun", "Mon", "Tue", "Wed", "Thu", "Fri", "Sat"]

    /// The next occurrence strictly after `completedDate`, or `nil` if that date would
    /// fall after `endDate` (inclusive) — meaning the recurrence is finished.
    /// Everything returned is start-of-day.
    func nextOccurrence(after completedDate: Date, calendar: Calendar = .current) -> Date? {
        let start = calendar.startOfDay(for: completedDate)
        let candidate: Date?
        switch frequency {
        case .everyNDays(let n):
            candidate = calendar.date(byAdding: .day, value: max(1, n), to: start)
        case .weekly(let days):
            candidate = Self.nextDay(after: start, calendar: calendar) { date in
                days.contains(calendar.component(.weekday, from: date))
            }
        case .monthly(let days):
            candidate = Self.nextDay(after: start, calendar: calendar) { date in
                days.contains(calendar.component(.day, from: date))
            }
        }
        guard let next = candidate else { return nil }
        if let end = endDate, next > calendar.startOfDay(for: end) { return nil }
        return next
    }

    /// Walk forward one day at a time from the day *after* `start` until `matches` is
    /// true. Capped at 400 days so a pathological/empty set can't loop forever.
    private static func nextDay(
        after start: Date,
        calendar: Calendar,
        matches: (Date) -> Bool
    ) -> Date? {
        var date = start
        for _ in 0..<400 {
            guard let next = calendar.date(byAdding: .day, value: 1, to: date) else { return nil }
            date = next
            if matches(date) { return date }
        }
        return nil
    }

    // MARK: - Persistence (JSON string, stored on TaskItem)

    func encoded() -> String? {
        guard let data = try? JSONEncoder().encode(self) else { return nil }
        return String(data: data, encoding: .utf8)
    }

    static func decode(from string: String) -> RecurrenceRule? {
        guard let data = string.data(using: .utf8) else { return nil }
        return try? JSONDecoder().decode(RecurrenceRule.self, from: data)
    }

    // MARK: - Display

    /// A short human summary, e.g. "Every day", "Every 3 days", "Mon · Wed · Fri",
    /// "1st of the month", "1st · 15th".
    var summary: String {
        switch frequency {
        case .everyNDays(let n):
            return n <= 1 ? "Every day" : "Every \(n) days"
        case .weekly(let days):
            let names = days.sorted().map { Self.weekdayShortNames[$0] }
            return names.joined(separator: " · ")
        case .monthly(let days):
            let sorted = days.sorted()
            if sorted.count == 1 {
                return "\(Self.ordinal(sorted[0])) of the month"
            }
            return sorted.map(Self.ordinal).joined(separator: " · ")
        }
    }

    /// "1st", "2nd", "3rd", "21st", … for a day-of-month.
    static func ordinal(_ n: Int) -> String {
        let suffix: String
        switch (n % 100, n % 10) {
        case (11, _), (12, _), (13, _): suffix = "th"
        case (_, 1): suffix = "st"
        case (_, 2): suffix = "nd"
        case (_, 3): suffix = "rd"
        default: suffix = "th"
        }
        return "\(n)\(suffix)"
    }
}
```

**Interfaces:**
- Consumes: `TaskItem.recurrenceRule`, `isRepeating`, `completionDates` (Task 2); `RecurrenceRule.nextOccurrence` (Task 1).
- Produces:
  - `static func finalizeOccurrence(_ task: TaskItem, in context: ModelContext, now: Date = Date())`
  - `clearCompleted(in:now:)` now reschedules repeats and deletes only non-repeating done tasks.
  - `DailyCleanup.runIfNeeded` passes its `now` into `clearCompleted`.

Behavior of `finalizeOccurrence(task, now)`:
1. `let completion = task.completedAt ?? now`; append `completion` to `task.completionDates`.
2. Clear the completed instance: `task.done = false`, `task.completedAt = nil`.
3. `next = rule.nextOccurrence(after: completion)`.
4. `next == nil` → `context.delete(task)` (recurrence over).
5. `next <= startOfDay(now)` → re-open in place: `task.isStashed = false`, `task.stashReturnDate = nil`.
6. else (future) → stash: `task.isStashed = true`, `task.stashReturnDate = next`.

    /// Finalize a done repeating task: append this completion to its history and move it
    /// to its next occurrence — re-opening in place (daily) or stashing until a future
    /// date (weekly/monthly). Deletes the task if it's past its end date.
    static func finalizeOccurrence(_ task: TaskItem, in context: ModelContext, now: Date = Date()) {
        guard let rule = task.recurrenceRule else { return }
        let completion = task.completedAt ?? now
        task.completionDates.append(completion)
        task.done = false
        task.completedAt = nil

        guard let next = rule.nextOccurrence(after: completion) else {
            context.delete(task)          // recurrence finished (past end date)
            try? context.save()
            return
        }
        let calendar = Calendar.current
        if next <= calendar.startOfDay(for: now) {
            task.isStashed = false        // due now → back in Today
            task.stashReturnDate = nil
        } else {
            task.isStashed = true         // waits in the stash's Repeating section
            task.stashReturnDate = next
        }
        try? context.save()
    }

    /// Delete all completed one-off tasks; reschedule completed *repeating* tasks instead
    /// of deleting them. Returns snapshots of what was actually removed (for undo) — a
    /// rescheduled repeat is not "removed" and is not included.
    @discardableResult
    static func clearCompleted(in context: ModelContext, now: Date = Date()) -> [TaskSnapshot] {
        let done = context.allTasks().filter { $0.done }
        for task in done where task.isRepeating {
            finalizeOccurrence(task, in: context, now: now)
        }
        return delete(done.filter { !$0.isRepeating }, in: context)
    }

import SwiftData

/// Edits a task's repeat schedule. Local pickers build a `RecurrenceRule`, committed to
/// the task on Save. Self-contained so a future task-info view can reuse it as a row.
struct RecurrenceEditorView: View {
    @Environment(\.modelContext) private var context
    @Environment(\.dismiss) private var dismiss
    @Bindable var task: TaskItem

    private enum Mode: String, CaseIterable, Identifiable {
        case everyNDays = "Every N days"
        case weekly = "Weekly"
        case monthly = "Monthly"
        var id: String { rawValue }
    }

    @State private var mode: Mode = .everyNDays
    @State private var interval: Int = 1
    @State private var weekdays: Set<Int> = [2]          // default Monday
    @State private var monthDays: Set<Int> = [1]         // default 1st
    @State private var hasEndDate = false
    @State private var endDate = Date()

    var body: some View {
        NavigationStack {
            Form {
                Section("Repeat") {
                    Picker("Frequency", selection: $mode) {
                        ForEach(Mode.allCases) { Text($0.rawValue).tag($0) }
                    }
                    .pickerStyle(.segmented)

                    switch mode {
                    case .everyNDays:
                        Stepper("Every \(interval) day\(interval == 1 ? "" : "s")",
                                value: $interval, in: 1...365)
                    case .weekly:
                        weekdayPicker
                    case .monthly:
                        monthDayPicker
                    }
                }

                Section("Ends") {
                    Toggle("End on a date", isOn: $hasEndDate)
                    if hasEndDate {
                        DatePicker("Ends", selection: $endDate, displayedComponents: .date)
                    }
                }

                if !task.completionDates.isEmpty {
                    Section("History") {
                        Text(historyLabel)
                            .foregroundStyle(Color.textSecondary)
                    }
                }

                if task.isRepeating {
                    Section {
                        Button(role: .destructive, action: removeRepeat) {
                            Text("Remove repeat")
                        }
                    }
                }
            }
            .navigationTitle("Repeat")
            .navigationBarTitleDisplayMode(.inline)
            .toolbar {
                ToolbarItem(placement: .cancellationAction) {
                    Button("Cancel") { dismiss() }
                }
                ToolbarItem(placement: .confirmationAction) {
                    Button("Save", action: save).disabled(!isValid)
                }
            }
            .onAppear(perform: loadFromTask)
        }
        .presentationDetents([.medium, .large])
    }

    private var weekdayPicker: some View {
        HStack {
            ForEach(1...7, id: \.self) { day in
                let on = weekdays.contains(day)
                Button {
                    if on { weekdays.remove(day) } else { weekdays.insert(day) }
                } label: {
                    Text(RecurrenceRule.weekdayShortNames[day].prefix(1))
                        .font(.subheadline.weight(.semibold))
                        .frame(width: 34, height: 34)
                        .background(Circle().fill(on ? Color.brand : Color.appSurface))
                        .foregroundStyle(on ? .white : Color.textPrimary)
                }
                .buttonStyle(.plain)
            }
        }
    }

    private var monthDayPicker: some View {
        let columns = Array(repeating: GridItem(.flexible(), spacing: 6), count: 7)
        return LazyVGrid(columns: columns, spacing: 6) {
            ForEach(1...31, id: \.self) { day in
                let on = monthDays.contains(day)
                Button {
                    if on { monthDays.remove(day) } else { monthDays.insert(day) }
                } label: {
                    Text("\(day)")
                        .font(.footnote)
                        .frame(maxWidth: .infinity, minHeight: 30)
                        .background(RoundedRectangle(cornerRadius: 6)
                            .fill(on ? Color.brand : Color.appSurface))
                        .foregroundStyle(on ? .white : Color.textPrimary)
                }
                .buttonStyle(.plain)
            }
        }
    }

    private var historyLabel: String {
        let count = task.completionDates.count
        guard let last = task.completionDates.max() else { return "Not done yet" }
        let when = last.formatted(date: .abbreviated, time: .omitted)
        return "Done \(count) time\(count == 1 ? "" : "s") · last on \(when)"
    }

    /// Weekly/monthly need at least one day selected to form a valid rule.
    private var isValid: Bool {
        switch mode {
        case .everyNDays: return true
        case .weekly: return !weekdays.isEmpty
        case .monthly: return !monthDays.isEmpty
        }
    }

    private func loadFromTask() {
        guard let rule = task.recurrenceRule else { return }
        switch rule.frequency {
        case .everyNDays(let n): mode = .everyNDays; interval = n
        case .weekly(let d): mode = .weekly; weekdays = d
        case .monthly(let d): mode = .monthly; monthDays = d
        }
        if let end = rule.endDate { hasEndDate = true; endDate = end }
    }

    private func save() {
        let frequency: Frequency
        switch mode {
        case .everyNDays: frequency = .everyNDays(interval)
        case .weekly: frequency = .weekly(weekdays)
        case .monthly: frequency = .monthly(monthDays)
        }
        task.recurrenceRule = RecurrenceRule(frequency: frequency, endDate: hasEndDate ? endDate : nil)
        try? context.save()
        Haptics.selection()
        Surfaces.reload()
        dismiss()
    }

    private func removeRepeat() {
        task.recurrenceRule = nil
        try? context.save()
        Haptics.impact(.light)
        Surfaces.reload()
        dismiss()
    }
}
